### Como executar

Clone a pasta do projeto, abra o terminal na pasta principal e execute os seguintes comandos. Também é possível ver o app rodando em [Links](#links).

```bash
  yarn  
  yarn dev
```

ou

```bash
  npm install 
  npm run dev
```
